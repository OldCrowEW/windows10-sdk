Detailed Description

Class to hold all configuration for the app. These settings are changeable by the Sensorberg backend. https://github.com/sensorberg-dev/android-sdk-samples/blob/master/technical-features.md#advanced-features Some settings are preset for the SDK but also custom data is possible.
